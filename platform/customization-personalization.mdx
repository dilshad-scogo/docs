## Embedding AI on Website

<Frame>
  <img src="/images/sia-logo-light.png" alt="Widget Embedding Guide" />
</Frame>

<AccordionGroup>
  <Accordion icon="code" title="Installation Code">
    Implementation steps:
    ```html
    <!-- SIA Widget Embed Code -->
    <script>
      window.SIAConfig = {
        widgetId: 'YOUR_WIDGET_ID',
        position: 'bottom-right',
        theme: 'light'
      };
    </script>
    <script src="https://widget.sia.ai/loader.js" async></script>
    ```
    
    <Warning>
      Place the script just before the closing body tag for optimal loading performance.
    </Warning>
  </Accordion>

  <Accordion icon="rocket" title="Performance Optimization">
    Best practices:
    - Async loading
    - Resource optimization
    - Cache management
    - Error handling
    
    <Note>
      The widget is designed to load without blocking your website's main content.
    </Note>
  </Accordion>
</AccordionGroup>

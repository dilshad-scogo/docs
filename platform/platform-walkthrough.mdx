## Introduction

Welcome to the SIA Platform Walkthrough! This guide covers the essential features of our platform, including creating and managing AI agents, utilizing the Knowledge Hub, and customizing platform settings. Follow along to make the most of your SIA experience.

## Agents

<Frame>
  <img src="/images/sia-logo-light.png" alt="Agent Creation Interface" />
</Frame>

<AccordionGroup>
  <Accordion icon="robot" title="Creating an Agent">
    Follow these steps to create your AI agent:
    1. Navigate to the Agents section
    2. Click "Create New Agent"
    3. Configure your agent settings:
       - Name and description
       - Response behavior
       - Access permissions
    
    <Frame>
      <img src="/images/sia-logo-light.png" alt="Agent Configuration Screen" />
    </Frame>
  </Accordion>
  
  <Accordion icon="pen-to-square" title="Editing an Agent">
    To modify existing agents:
    1. Select the agent from your dashboard
    2. Click "Edit" or the settings icon
    3. Update the desired settings
    4. Save your changes

    <Tip>
      Regular updates to your agent's settings can help optimize its performance and accuracy.
    </Tip>
  </Accordion>

  <Accordion icon="code" title="Website Integration">
    Embed your agent on any website:
    1. Go to the agent's settings
    2. Find the "Integration" section
    3. Copy the provided snippet code
    4. Paste the code into your website's HTML

    <CodeGroup>
      <Code title="HTML Snippet">
        ```html
        <!-- SIA Agent Embed Code -->
        <div id="sia-agent"></div>
        <script src="[Your-Agent-URL]"></script>
        ```
      </Code>
    </CodeGroup>
  </Accordion>
</AccordionGroup>

## Knowledge Hub

<Frame>
  <img src="/images/sia-logo-light.png" alt="Knowledge Hub Dashboard" />
</Frame>

<CardGroup>
  <Card title="Document Upload" icon="file-arrow-up">
    Upload process:
    - Click "Upload Documents"
    - Select PDF or DOC files
    - Wait for processing completion
    - Verify successful upload
  </Card>

  <Card title="Access Levels" icon="lock">
    Available access levels:
    - Team Member: Limited to specific team members
    - Entire Organization: Available to all org members
    - Public: Accessible to everyone
  </Card>

  <Card title="Knowledge Base Integration" icon="link">
    Connect to agents:
    - Select target agent
    - Choose knowledge base
    - Set access permissions
    - Confirm integration
  </Card>
</CardGroup>

## Platform Features & Settings

    - Submit feedback forms
    - Contact support team
    - Track support tickets
  </Accordion>

  <Accordion icon="arrow-right-from-bracket" title="Account Security">

    1. Click your profile icon
    2. Select "Logout"

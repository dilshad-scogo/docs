## Introduction

Welcome to the SIA Platform Walkthrough! This guide covers the essential features of our platform, including creating and managing AI agents, utilizing the Knowledge Hub, and customizing platform settings. Follow along to make the most of your SIA experience.

## Agents

<Frame>
  <img src="/images/platform/agents.png" alt="Agent Creation Interface" />
</Frame>

<AccordionGroup>
  <Accordion icon="robot" title="Creating an Agent">
    Follow these steps to create your AI agent:
    1. Navigate to the Agents section
    2. Click "Create Agent"
    3. Configure your agent settings:
       - Domain
       - Description
       - Logo URL
    4. Click on Create Button and you AI will be ready to support your customer. 
    
    
    <Frame>
      <img src="/images/platform/agent-created.png" alt="Agent Screen" />
    </Frame>

  </Accordion>
  
  <Accordion icon="pen-to-square" title="Editing an Agent">
  <Frame>
    <img src="/images/platform/edit-agent.png" alt ="Edit Agent" />
  </Frame>
    To modify existing agents:
    1. Click "Edit" or the settings icon
    2. Update the desired settings
    3. Save your changes
 
  </Accordion>

  <Accordion icon="code" title="Website Integration">
    Embed your agent on any website:
    1. Go to the agent's settings
    2. Find the "Integration" section
    3. Copy the provided snippet code
    4. Paste the code into your website's HTML

    <CodeGroup>
      <Code title="HTML Snippet">
        ```html
        <!-- SIA Agent Embed Code -->
        <div id="sia-agent"></div>
        <script src="[Your-Agent-URL]"></script>
        ```
      </Code>
    </CodeGroup>
  </Accordion>
</AccordionGroup>

## Knowledge Hub

<Frame>
  <img src="/images/platform/kb.png" alt="Knowledge Hub Dashboard" />
</Frame>
A folder in Knowledge Hub will be ctreated against your AI Agent which you created.
All the knowledge provided in this folder will be mapped with that AI Agent after providing the access. 
<AccordionGroup>
  <Accordion title="Document Upload" icon="file-arrow-up">
    <Frame>
    <img src ="/images/platform/kb-folder.png" alt="Knowledge Base Folder" />
    </Frame>
    Upload process:
    - Navigate to Knowledge Hub
    - Select the desired folder. 
      
    - Click on File Upload and Select the file.
    <Frame>
      <div style={{ display: 'flex', justifyContent: 'space-between' }}>
        <img src="/images/platform/file-upload.png" alt="Upload File" height="200" width="200" style={{ marginRight: '80px' }} />
        <img src="/images/platform/select-file.png" alt="Upload File" height="180" width="180" />
      </div>
    </Frame>
    - File will be uploaded and will be visible on the knowledge hub.
    <img src="/images/platform/file-uploaded.png" alt="File Uploaded" height="300" width="700" />

  </Accordion>

  <Accordion title="Access Levels" icon="lock">
  Here you can manage the access level of the document.
  <img src="/images/platform/access-level.png" alt="Access Level" height="180" width="180"/>
  
    Available access levels:
    - Just Me: Accessible for indiviual 
    - Team Members : Limited to specific team.
    - Entire Organization: Available to all org members
    - Extended Access: Accessible via Ask AI Agent and AI Call. 
  </Accordion>

  <Accordion title="Knowledge Base Integration" icon="link">
    Connect to agents:
    - Navigate to target file in knowledge hub.
    - Set access permissions to Extended Access.

    AI Agent will answer from the the provided document.
  </Accordion>
</AccordionGroup>
